SDK Components Installation Guide

System Requirements

This kit will require the following system specifications:

  • Minimum of Pentium III 500 MHz or Higher
  • Microsoft Windows 2000 / XP / Vista
  • Minimum of 128 MB RAM
  • Minimum of 600 MB Free Hard Disk Space
  • CD-ROM Drive
  • USB Port
  • Serial RS232 port

SDK Checklist

The ACR122S NFC Reader SDK contains the following:

  • 1 ACR122S NFC Reader
  • 5 Mifare 1K Cards
  • 3 Topaz Tags
  • 1 ACOS6-SAM Card
  • SDK CD-ROM

Connection Diagram

To connect the ACR122S reader to your PC, plug in the 9-PIN serial connector to the Serial/COM port and the USB connector to an available USB port.

Install Smart Card Reader Driver

ACR122S does not require any driver installation as it uses the generic COM driver, which is already installed in Windows and Linux OSes.

Install SDK

Insert the ACR122S SDK CD-ROM to your CD-ROM drive. If the screen below does not appear, run x:\AUTORUN.EXE where x is the drive letter of your CD-ROM.  

SDK Components

The default path where the SDK will be installed is x:\Program Files\Advanced Card Systems Ltd\ACR122S Serial NFC Reader SDK where x is the drive letter of your local Windows drive. Click here to view the SDK Installation Guide.

Sample Codes - The sample codes show the different capabilities of the ACR122S and showcases how to control the ACR122S peripherals and how to communicate with a contactless tag and SAM.

The sample codes are written in the following programming languages: Java, Delphi 7, Visual Basic 6, Visual C++ 6, Visual C++ 2005 (x64), Visual C# 2005 and Visual Basic .Net 2005.

Tools and Utilities

ACR122S Tool – The ACR122S Tool is an application utility tool that enables the user to perform reader and card-related commands.

ACS Easy Key - The ACS Easy Key is a utility program that allows you to change Mifare security settings.

QuickView - The ACS QuickView is a utility program that checks if you have properly installed the ACR122S device.
